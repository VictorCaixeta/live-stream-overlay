URLs para OBS Studio

Adicione estas URLs como Browser Sources no OBS

🖥️ Browser Sources
📖 Como Configurar no OBS
  1. Abra o OBS Studio
  2. Na seção "Fontes", clique em + → Navegador
  3. Dê um nome (ex: "Overlay Principal")
  4. Cole a URL correspondente no campo URL
  5. Para o overlay principal: Largura: 1920, Altura: 1080
  6. Para o chat: Largura: 400, Altura: 600
  7. Para alertas: Largura: 1920, Altura: 1080
  8. Marque "Desligar fonte quando não visível"
  9. Clique em OK
